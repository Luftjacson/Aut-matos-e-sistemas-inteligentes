# README — Jogos interativos: Sistemas inteligentes e autômatos no campo

## 1. Identificação do material

**Título:** Jogos interativos — Sistemas inteligentes e autômatos no campo  
**Professor:** Jacson Luft  
**Área:** Robótica e Tecnologia  
**Formato:** Página HTML interativa, com geração de relatório para impressão ou salvamento em PDF.

Este HTML foi desenvolvido para aplicação de atividades gamificadas sobre sistemas inteligentes, automação, sensores, atuadores, Arduino e situações-problema relacionadas ao campo.

## 2. Objetivo pedagógico

O material tem como objetivo revisar e avaliar, de forma interativa, os principais conceitos relacionados a sistemas inteligentes e autômatos, especialmente:

- entrada, processamento e saída de dados;
- sensores e atuadores;
- Arduino como controlador;
- sequência lógica de funcionamento de sistemas automatizados;
- aplicação da automação em situações reais do cotidiano e do campo;
- interpretação de problemas tecnológicos.

## 3. Estrutura da atividade

O HTML é composto por cinco jogos interativos e uma proposta de projeto final.

### Jogos interativos

1. **Jogo da memória: componente e função**  
   O estudante deve relacionar componentes tecnológicos às suas respectivas funções.

2. **Classificação: entrada, processamento e saída**  
   O estudante deve organizar os elementos do sistema nas categorias corretas.

3. **Sequência lógica**  
   O estudante deve ordenar as etapas de funcionamento de um sistema inteligente.

4. **Missões-problema**  
   O estudante responde a situações-problema envolvendo automação e tomada de decisão.

5. **Lacunas conceituais**  
   O estudante completa frases com os conceitos adequados.

### Projeto final

O projeto final vale 30 pontos e será avaliado pelo professor após a entrega no Classroom.

## 4. Pontuação

A atividade totaliza **100 pontos**, distribuídos da seguinte forma:

| Etapa | Pontuação |
|---|---:|
| Jogo da memória | 14 pontos |
| Classificação | 14 pontos |
| Sequência lógica | 14 pontos |
| Missões-problema | 14 pontos |
| Lacunas conceituais | 14 pontos |
| Projeto final | 30 pontos |
| **Total** | **100 pontos** |

A pontuação dos jogos interativos compõe a **pontuação geral**, totalizando até **70 pontos**.

O projeto vale **30 pontos** e será avaliado posteriormente pelo professor, considerando a entrega realizada no Classroom.

## 5. Campos de identificação

No início da atividade, o estudante deve preencher:

- nome;
- turma;
- data;
- número da chamada.

No relatório impresso, também consta o nome do professor: **Professor Jacson Luft**.

## 6. Impressão e geração de PDF

Ao final da atividade, o estudante deve clicar no botão de impressão ou salvamento em PDF.

O PDF gerado apresenta:

1. identificação do estudante;
2. pontuação geral;
3. relatório da atividade;
4. respostas das atividades realizadas pelo aluno;
5. observação sobre o projeto final, que será avaliado pelo professor após a entrega no Classroom.

## 7. Orientações para uso em sala de aula

Sugere-se que o professor oriente os estudantes a:

1. preencher corretamente os dados de identificação;
2. realizar os jogos na ordem indicada;
3. ler atentamente os feedbacks apresentados;
4. revisar as respostas antes de gerar o PDF;
5. salvar ou imprimir o relatório final;
6. entregar o projeto no Classroom, conforme orientação do professor.

## 8. Recomendações técnicas

Para melhor funcionamento, recomenda-se:

- abrir o arquivo em navegador atualizado, como Google Chrome, Microsoft Edge ou Firefox;
- não editar o código durante a aplicação da atividade;
- testar a impressão em PDF antes da aplicação com a turma;
- orientar os alunos a gerar o PDF somente após concluir todas as atividades.

## 9. Observação sobre avaliação

A pontuação automática do HTML refere-se apenas aos jogos interativos.  
A pontuação do projeto não é lançada automaticamente no HTML, pois depende da análise do professor após a entrega no Classroom.

## 10. Autoria e finalidade

Material elaborado para uso pedagógico em atividades de Robótica e Tecnologia, com foco em aprendizagem ativa, gamificação e aplicação contextualizada dos conceitos de sistemas inteligentes no campo.
